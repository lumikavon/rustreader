# RustReader 用户手册

## 简介

RustReader 是一款轻量级的桌面文件浏览器和阅读器，支持预览多种常见文件格式。基于 Tauri 2.x 构建，提供跨平台支持（Windows、macOS、Linux）。

---

## 安装

### Windows

1. 下载 `.msi` 或 `.exe` 安装包
2. 双击运行安装程序
3. 按照向导完成安装
4. 从开始菜单或桌面快捷方式启动应用

### macOS

1. 下载 `.dmg` 安装包
2. 双击打开 DMG 文件
3. 将 RustReader 拖入「应用程序」文件夹
4. 从启动台或应用程序文件夹启动

### Linux

**Debian/Ubuntu (.deb)**：
```bash
sudo dpkg -i rustreader_1.0.0_amd64.deb
```

**AppImage**：
```bash
chmod +x rustreader_1.0.0_amd64.AppImage
./rustreader_1.0.0_amd64.AppImage
```

## 功能说明

### 1. 打开文件/文件夹

点击顶部导航栏的 **「打开」** 下拉菜单：

| 选项 | 说明 |
|------|------|
| **打开文件夹...** | 选择一个文件夹，加载其中的所有可预览文件 |
| **打开文件...** | 选择一个或多个文件进行预览 |
| **打开最近** | 快速访问最近打开的文件夹或文件 |

### 2. 文件类型筛选

顶部导航栏右侧显示文件类型筛选按钮，点击可快速筛选显示特定类型的文件：

| 类型 | 支持的格式 |
|------|-----------|
| **图片** | `.png` `.jpg` `.jpeg` `.gif` `.webp` |
| **视频** | `.mp4` `.webm` `.ogv` `.m4v` |
| **音频** | `.mp3` `.wav` `.m4a` `.ogg` `.flac` `.aac` |
| **Markdown** | `.md` `.markdown` |
| **思维导图** | `.mm.md`（特殊命名的 Markdown） |
| **演示文稿** | `.pptx`、`.ppt.md`（Marpit 格式） |
| **PDF** | `.pdf` |
| **Word** | `.docx` |
| **Excel** | `.xlsx` |
| **文本** | `.txt` |
| **Draw.io** | `.drawio` |

### 3. 目录树

左侧目录树显示当前加载的文件结构：

- **搜索框**：输入关键词快速搜索文件名
- **文件夹**：点击展开/折叠
- **文件**：点击在右侧预览区显示

### 4. 文件预览

右侧预览区支持多种文件的在线预览：

| 功能按钮 | 说明 |
|---------|------|
| **↑ / ↓** | 切换到上一个/下一个文件 |
| **全屏** | 全屏显示当前预览内容 |
| **侧边栏切换** | 显示/隐藏左侧目录树 |

**图片预览特殊操作**：
- **滚轮**：缩放图片
- **拖拽**：平移图片
- **双击**：重置缩放

### 5. 语言切换

点击顶部的 **「EN」** 或 **「中文」** 按钮切换界面语言（支持中文和英文）。

### 6. GitHub 链接

点击 GitHub 图标可访问项目开源仓库：
https://github.com/lumikavon/rustreader

### 7. 关于

点击 **「关于」** 按钮查看软件信息：
- 软件名称
- 版本号
- 作者
- 微信
- GitHub 地址

## 命令行使用

RustReader 支持命令行参数启动：

```bash
# 直接打开文件夹
rustreader /path/to/folder

# 直接打开文件
rustreader /path/to/file.md

# 使用 --open 或 -o 参数
rustreader --open /path/to/folder
rustreader -o /path/to/file.pdf

# 自定义应用标题
rustreader --site-name "我的文档库" /path/to/folder
```

## 常见问题

### Q: 为什么某些文件无法预览？

A: RustReader 目前支持特定格式的文件预览。不支持的文件类型会显示在目录树中，但无法在预览区显示内容。

### Q: 如何更改默认字体大小？

A: 使用 `Ctrl + =` 放大或 `Ctrl + -` 缩小字体。设置会自动保存，下次启动时生效。

### Q: 应用启动后是空白的？

A: 请使用「打开」菜单选择一个文件夹或文件。首次启动时需要手动选择要浏览的内容。

### Q: 如何卸载？

**Windows**：通过「控制面板」→「程序和功能」卸载

**macOS**：将应用拖入废纸篓

**Linux (deb)**：
```bash
sudo dpkg -r rustreader
```
